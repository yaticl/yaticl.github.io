Maximum-Length Sequence

簡介

Maximum-Length Sequence (MLS) 是一種二值序列，長度\(P=2^N-1\)，其中\(N\)代表整數。MLS 可以利用以下的關係式遞迴產生：

$$n(k+3)=n(k) \oplus n(k+2),$$

其中\(\oplus\)代表 XOR 運算。若將將產生出來的\(0,1\)邏輯值由下式轉換成\(1,-1\)，則為 MLS 訊號：

Hadamard Matrix

Posted on Tue 06 March 2018 in Linear algebra • Tagged with Linear algebra, Digital signal processing, Python

簡介

Hadamard matrix 是一種只包含\((-1,+1)\)的矩陣，其特性是取任兩行或是列放在一起比較，則會有一半是同號，另一半則是不同號。
當要畫出該矩陣時，定義\(+1\)代表黑色，而\(-1\)代表白色。

特性

\({n \times n}\)的 Hadamard matrix \(H_n\)會有\(n \frac{n(n-1)}{2}\)個\((-1)\)方塊以及\(\frac{n(n+1)}{2}\)個\((+1)\)的方塊。
